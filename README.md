# শুভ্র টেলিকম — ইনভেন্টরি ম্যানেজমেন্ট সিস্টেম

কম্পিউটার ও ইলেকট্রিক্যাল শপের জন্য প্রিমিয়াম, Apple-অনুপ্রাণিত, গ্লাসমরফিজম ডিজাইনের ইনভেন্টরি ম্যানেজমেন্ট সিস্টেম।

## 🛠️ স্ট্যাক

- **React 18 + TypeScript + Vite**
- **Tailwind CSS** (কাস্টম ডিজাইন টোকেন / CSS ভেরিয়েবল ভিত্তিক লাইট-ডার্ক থিম)
- **Framer Motion** — মোডাল, টোস্ট, পেজ ট্রানজিশন অ্যানিমেশন
- **React Hook Form + Zod** — টাইপ-সেফ ফর্ম ভ্যালিডেশন
- **Recharts** — ড্যাশবোর্ড চার্ট
- **Lucide React** — আইকন
- **SheetJS (xlsx)** — এক্সেল ইম্পোর্ট/এক্সপোর্ট/টেমপ্লেট

## 🚀 লোকাল মেশিনে চালানো

```bash
# ১. ডিপেন্ডেন্সি ইনস্টল করুন
npm install

# ২. ডেভেলপমেন্ট সার্ভার চালু করুন
npm run dev

# ৩. ব্রাউজারে http://localhost:5173 খুলুন
```

## 📦 প্রোডাকশন বিল্ড

```bash
npm run build     # dist/ ফোল্ডারে অপ্টিমাইজড বিল্ড তৈরি হবে
npm run preview   # বিল্ড লোকালি প্রিভিউ করুন
```

## 📁 ফোল্ডার স্ট্রাকচার

```
src/
  types/          → TypeScript ইন্টারফেস (Product, Category, Brand, Supplier, StockTx...)
  data/seed.ts     → ডেমো ডেটা (৪৪ টি পণ্য, ১০ ক্যাটাগরি, ২০ ব্র্যান্ড, ৭ সাপ্লায়ার)
  lib/
    utils.ts       → ফরম্যাটার (৳, তারিখ, সংখ্যা) ও হেল্পার
    excel.ts       → SheetJS টেমপ্লেট/ইম্পোর্ট/এক্সপোর্ট লজিক
    validation.ts  → Zod স্কিমা (Product, Supplier, Category, Brand, Stock forms)
  context/
    ToastContext.tsx → গ্লোবাল টোস্ট নোটিফিকেশন সিস্টেম
  components/
    ui/            → পুনঃব্যবহারযোগ্য প্রিমিটিভ (Modal, Badge, Select, Pagination...)
    layout/        → Sidebar, Topbar, নেভিগেশন কনফিগ
    shared/        → ProductAvatar
    products/      → ProductFormModal, BarcodeModal, ImportExcelModal
  pages/           → প্রতিটি মডিউলের পূর্ণাঙ্গ পেজ (Dashboard, Products, Categories, ...)
  App.tsx          → রুট শেল — স্টেট, রাউটিং, কীবোর্ড শর্টকাট
  main.tsx         → এন্ট্রি পয়েন্ট
```

## ✨ মডিউল

- **ড্যাশবোর্ড** — KPI কার্ড, স্টক মুভমেন্ট চার্ট, ক্যাটাগরি-ভিত্তিক পাই চার্ট, লো-স্টক ও সাম্প্রতিক কার্যক্রম
- **বিক্রয় / ক্যাশ মেমো** — মাল্টি-আইটেম ইনভয়েস তৈরি, কাস্টমার বা ওয়াক-ইন নির্বাচন, ডিসকাউন্ট, আংশিক/সম্পূর্ণ পেমেন্ট, স্বয়ংক্রিয় স্টক হ্রাস, প্রিন্টযোগ্য ক্যাশ মেমো
- **ক্রয়** — সাপ্লায়ার থেকে মাল্টি-আইটেম ক্রয়, স্বয়ংক্রিয় স্টক বৃদ্ধি, বাকি/পরিশোধ ট্র্যাকিং
- **পণ্য** — সার্চ, ফিল্টার, সর্ট, পেজিনেশন, বাল্ক সিলেক্ট/ডিলিট, বারকোড লেবেল, ইম্পোর্ট/এক্সপোর্ট
- **ক্যাটাগরি / ব্র্যান্ড / সাপ্লায়ার** — সম্পূর্ণ CRUD
- **স্টক ইন/আউট** — ট্রানজেকশন ফর্ম + ইতিহাস
- **স্টক সমন্বয়** — ফিজিক্যাল কাউন্ট রিকনসিলিয়েশন
- **লো স্টক এলার্ট** — লো-স্টক ও আউট-অফ-স্টক ট্যাব
- **দৈনিক আয়-ব্যয় (ক্যাশবুক)** — ম্যানুয়াল আয়/ব্যয় এন্ট্রি, বিক্রয়/ক্রয় থেকে স্বয়ংক্রিয় এন্ট্রি, রানিং ব্যালেন্স, **"হিসাব শুরুর ব্যালেন্স" (Opening Balance)** — পুরানো খাতা থেকে যেকোনো তারিখ ও পরিমাণ দিয়ে হিসাব শুরু/স্থানান্তর করার সুবিধা
- **কাস্টমার** — কাস্টমার তালিকা, প্রতিটির জন্য **"পূর্বের বকেয়া" (Opening Due)** ফিল্ড দিয়ে পুরানো হিসাব মাইগ্রেট করা যায়, সম্পূর্ণ লেজার (ইনভয়েস + পেমেন্ট + রানিং ব্যালেন্স), পেমেন্ট গ্রহণ
- **রিপোর্ট** — ৭ ধরনের রিপোর্ট, এক্সেল এক্সপোর্ট, প্রিন্ট/PDF
- **সেটিংস** — দোকানের তথ্য, ডার্ক মোড, অটো-সেভ, **ব্যাকআপ ডাউনলোড/পুনরুদ্ধার (JSON)**, ডেমো রিসেট, কীবোর্ড শর্টকাট

## 💾 ডেটা কীভাবে সংরক্ষিত হয়

- সব তথ্য (পণ্য, বিক্রয়, ক্রয়, কাস্টমার, ক্যাশবুক...) প্রতি পরিবর্তনে স্বয়ংক্রিয়ভাবে ব্রাউজারের `localStorage`-এ সংরক্ষিত হয় — পেজ রিফ্রেশ বা বন্ধ করলেও হারাবে না।
- **সেটিংস → ব্যাকআপ ও পুনরুদ্ধার** থেকে যেকোনো সময় সম্পূর্ণ ডেটার একটি `.json` ফাইল ডাউনলোড করতে পারবেন (অন্য কম্পিউটারে স্থানান্তর বা নিরাপদ কপি রাখতে), এবং সেই ফাইল থেকে যেকোনো সময় সম্পূর্ণ পুনরুদ্ধার করা যাবে।
- **দৈনিক আয়-ব্যয় → ওপেনিং ব্যালেন্স** থেকে আপনি যেকোনো তারিখ ও নগদ পরিমাণ দিয়ে হিসাব শুরু করতে পারবেন — অর্থাৎ আগের খাতার হিসাব এখানে একটি সংখ্যা হিসেবে বসিয়ে সেদিন থেকেই সিস্টেমে হিসাব রাখা শুরু করা যায়।
- একইভাবে **কাস্টমার** যোগ করার সময় "পূর্বের বকেয়া" ফিল্ডে খাতার পুরানো বকেয়া বসিয়ে দিলে সেটিও নতুন হিসাবের সাথে যোগ হয়ে যাবে।

## ⌨️ কীবোর্ড শর্টকাট

| কী | কাজ |
|---|---|
| `/` | সার্চ বক্সে ফোকাস |
| `N` | নতুন পণ্য (পণ্য পাতায়) |
| `Esc` | মোডাল বন্ধ |
| `G` তারপর `D` | ড্যাশবোর্ড |
| `G` তারপর `P` | পণ্য তালিকা |

## 📊 এক্সেল ইম্পোর্ট ফরম্যাট

`পণ্য` পাতা থেকে "টেমপ্লেট ডাউনলোড করুন" বাটনে ক্লিক করলে সঠিক কলাম হেডারসহ একটি `.xlsx` ফাইল পাবেন। SKU বা বারকোড মিলে গেলে বিদ্যমান পণ্য আপডেট হয়, না মিললে নতুন পণ্য তৈরি হয়। ক্যাটাগরি/ব্র্যান্ড/সাপ্লায়ার নাম না থাকলে স্বয়ংক্রিয়ভাবে তৈরি হয়।

## 🎨 থিম কাস্টমাইজেশন

সব রং `src/index.css`-এর `:root` ও `.dark` ব্লকে CSS ভেরিয়েবল হিসেবে সংজ্ঞায়িত, এবং `tailwind.config.js`-এ Tailwind ইউটিলিটি ক্লাস (`bg-accent`, `text-danger` ইত্যাদি) হিসেবে ম্যাপ করা। রং পরিবর্তন করতে শুধু `index.css`-এর ভেরিয়েবলগুলো পরিবর্তন করুন।
